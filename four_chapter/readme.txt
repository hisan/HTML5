一：重要概念
	1.css:层叠样式表，用来规定HTML文档的呈现形式（外观和格式编排）


二：css属性汇总：

backgroud-color		设置元素的背景颜色

border				设定围绕元素的边框

color  				设置元素的前景颜色

font-size			设置元素文字的字号

height				设置元素高度

padding				设置元素内容和边框的间距

text-decoration		设置元素文字的装饰效果

width				设置元素宽度


三：操作
	1.样式的操作：定义和使用。
	
	2.全局属性style：将css当作其元素的属性

	3.使用文档内嵌样式：
		3.1 针对于大量需要各种样式的复杂文档（不必逐个元素涉及css,软件更新时也不用逐个元素修改）
		
		3.2	用style元素（而不是属性）定义文档内嵌样式，通过css选择器指示浏览器应用样式。
			3.2.1 一个style元素可以定义多条样式。

	4.使用外部样式表：针对于一套样式，用于多个HTML文档,则可以增加一个样式表文件.css.
		4.1 文档想链接多少样式表都行，每个样式表用一个link元素说明。
		4.2 样式表互相关联(将一个样式表导入另一个样式表)：@import "other.css";
			4.2.0 @import语句只能出现在文档开头，能在它上面的语句只能是@charset "UTF-8";
			4.2.1 若在互相导入的过程中出现重复样式，则“最下面的”覆盖“最上面的”;
			4.2.2 声明样式表的编码
