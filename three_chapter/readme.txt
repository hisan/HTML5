第三章 初探HTML


一：重要概念

1.class 对元素进行分类，以便统一其样式或用程序查找该元素。

2.id:为元素分配一个独一无二的标识符，以便对其应用样式或程序选择该元素。

3.style:直接定义元素样式


标签：标签只是解读HTML文档，不会显示。(如今，你们反对使用标签改变HTML元素的样式，这应该由css来完成)


4.虚元素：如 hr，，只能使用一个标签表示，在其中放置任何内容都不符合HTML规范。

5.注释 <!-- text -->


二.元素汇总


a 				生成超链接
body 			表示HTML文档的内容
button 			生成可以提交表达的按钮

code 			表示计算机代码片段
DOCTYPE			表示HTML文档的开始
head			表示HTML文档的头部区域
hr				表示主题的改变
html            表示文档的HTML部分
input			表示用户输入的数据
label 			生成另一元素的说明标签
p				表示段落
style			表示css样式
table			表示用表格组织的数据
td				表示表格单元格
textarea 		生成用于获取用户输入数据的多行文本框
th				生成表头单元格
title			表示HTML文档的标题
tr 				表示表格行


三.元素属性：（属性名：值）
	1.元素可以用属性进行配置。
	2.属性只能在开始标签或单个标签上，不能用于结束标签。
	3.有一些全局属性可用于所有HTML元素，还有用来提供其特有配置信息的专有属性。
	4.一个元素可以有多个属性，属性间以一个或多个空格分割即可。属性间无顺序要求。
	5.bool属性：不需要设定值，只需要将属性名添加到元素中即可。即：元素：属性名
	6.用户自定义属性(扩展属性)：必须以data-开头，如data-creator="adam" data-purpose="collection"

四：元素详细说明
	1.外层结构：DOCTYPE
	2.元数据：向浏览器提供文档的信息.如 <title></title>
	3.内容：<body></body>
	4.元素之间的父子关系：
		4.2：子元素是关系最近的后代元素。
		4.1：一个元素可以有多个子元素，但是只能有一个父元素。
	5.元素类型：元数据元素、流元素、短语元素

五：HTML实体（为了将那些具有特殊意义，如<与>不被当做HTML处理，为此应使用HTML实体）
	部分实体表示：
		&lt == < 
		&gt == >

六：HTML5全局属性：
	1.accesskey：用于设定一个或多个用于选择页面上的元素的快捷键。

	2.class属性：用于将元素归类。这样通常是为了能够找出文档中某一类元素或为某一类元素应用CSS样式。
		2.1 一个元素可以归为多种类别，为此在class属性的属性值中使用空格隔出即可各个类名即可。
		2.2 class属性的一种利用方式是设计css样式时，以所定义的一个或多个类作为应用目标。

	3.contenteditable:让用户能修改页面上的内容

	4.contextmenu:用来为元素设定快捷键菜单。

	5.dir:用来规定元素中文字的方向：ltr(用于从左到右的文字),rtl(用于从右到左的文字)	

	6.draggable：用来表示元素是否可拖放。

	7.dropzone:与draggable搭配使用

	8.hidden:一个布尔属性，表示相关元素当前不需要关注，浏览器对它的处理方式是隐藏相关元素。

	9.id:用来给元素分配一个唯一的标识符。这种标识符通常用来将样式应用到元素上、或在js程序中用来选择元素。

	10.lang属性：说明元素使用的语言。lang属性值必须使用有效的ISO语言代码。

	11.spellcheck属性:用来表明浏览器是否应该对元素的内容进行拼写检查。这个属性只有在用户可以编辑的元素上才有意义。

	12.style：用于直接在元素身上定义css样式

	13.tabindex:HTML页面上的键盘焦点可以通过按Tab键在各元素之间切换。用Tabindex属性可以改变默认的转移顺序。

	14.title:提供了元素的额外信息，浏览器通常用这些东西显示工具提示。

七：个人总结
	1：css，元素的一种属性集合，作用即是用于呈现元素样式。
	2:script:js,肉眼可见的交互。控制和改变元素的属性。
